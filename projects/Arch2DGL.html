Arch2DGL

2D Java and OpenGL Game Library

Arch2DGL

Arch2DGL is a 2D java game framework that is based on OpenGL using the LWJGL which provides direct access to OpenGL. We also provide documentation and tutorials on how to use our framework here.

Arch2DGL also provides simple methods and interfaces that are easy and intuitive utilizing low level OpenGL and GLFW. The framework is ideal for newer programmers that are looking to jump right into game/2D development but aren't willing to learn the ins and outs of OpenGL.

The library is open source and free to use and feel free to contribute to the code and development here.
