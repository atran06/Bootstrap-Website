Bounce: 2D game in which the player controls a ball to reach the end

   Febuary 10, 2017

About

Bounce is a 2D game made in Java in which you control a ball that you can manipulate to change direction in order to reach the golden door. Gameplay utilizes four keys and currently has 5 levels to play with increasing difficulty.
