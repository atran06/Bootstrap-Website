About

Bounce is a 2D game I made in java that is essentially a game in which the player manipulates the ball's direction to guide it to the end of the level. As of now, the game has 5 different levels that you can play through.

Features

  • Multiple Levels
  • Different Ball Skins
  • Custom Soundtrack
  • Simple Controls
